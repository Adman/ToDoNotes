pyToDo
==========
Remember your tasks via terminal, show them and remove them when they are done!

Dependencies
------------
* Python 2.x+
* Pynotify

Usage
-----
./pytodo.py [option] [task]

options can be:
* add - adds a task
* rm/remove/done - gets the task done and removes it

or run just ./pytodo.py and see all the tasks
